This plugin can show the posts (and pages) that have been recently viewed on your wordpress blog.

== Description ==

This plugin displays the posts (and pages) that have been recently viewed by the visitors. The recently 
viewed posts are saved in a the database and by default the last 10 visited posts are displayed in the sidebar
or on a page with the [recently] shortcode.

== Installation ==

1. Install Recently Viewed Posts either via the WordPress.org plugin directory, or by uploading the files to your server
2. After activating Recently Viewed Posts, you can use the widget on your sidebar or add the shortcode [recently] on a post or page.
5. That's it.  You're ready to go!
